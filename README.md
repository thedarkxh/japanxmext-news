## 🚀 Quick Start
1. **Get your API Key:** Sign up at the [Groq Cloud Console](https://console.groq.com/).
2. **Configure:** Paste your key into the `GROQ_API_KEY` variable in `japan_scout.py`.
3. **Install Dependencies:** `pip install requests beautifulsoup4`
4. **Run:** `python japan_scout.py`
5. **Monitor:** Check `INTEL_REPORT_[TIMESTAMP].log` for your 10-minute brief.
